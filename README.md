Simple blog made with nuxt & bulma

## Build Setup

### 事前準備
1. インターネット環境
2. [Git](https://git-scm.com/)
3. [nodejs](https://nodejs.org/)
4. （任意）[yarn](https://yarnpkg.com/)

### コマンドを実行
```bash
git clone https://github.com/Osumi-Akari/nuxtblog-miller.git
cd nuxt-miller
yarn install
yarn generate
```

distフォルダが生成されたことを確認
