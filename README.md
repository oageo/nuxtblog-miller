## Build Setup

### 事前準備
1. インターネット環境
2. [Git](https://git-scm.com/)
3. [nodejs](https://nodejs.org/)
4. （任意）[pnpm](https://pnpm.io/ja/)

### コマンドを実行
```bash
git clone https://github.com/oageo/nuxtblog-miller.git
cd nuxt-miller
pnpm install
pnpm generate
```

distフォルダが生成されたことを確認
